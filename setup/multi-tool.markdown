Mapbase has an external utility program used for miscellaneous Mapbase setup tasks.

At the moment, it has two functions: **Creating tool shortcuts** and **Converting VMT files to the new shaders**.

<p align="left">
<img src="{{ "/" | relative_url }}assets/setup/multitool.png" width="384"/>
</p>

## Shortcut Creator

This tool automatically creates `-game` shortcuts to the SDK tools so they could use Mapbase's shaders. This is meant to be a quick and easy alternative to manually creating new shortcuts with the `-game` launch parameter.

<p align="left">
<img src="{{ "/" | relative_url }}assets/setup/multitool-shortcuts.png" width="768"/>
</p>

## Asset Converter

This tool converts entire directories of VMT files to Mapbase's custom `SDK_` shaders. It is extremely important for mods which want to use Mapbase and have tons regular VMT files which should start using the new shades. 

<p align="left">
<img src="{{ "/" | relative_url }}assets/setup/multitool-assets.png" width="768"/>
</p>
