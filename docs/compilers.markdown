Mapbase primarily requires custom map compilers for the following reasons:

* Mapbase uses custom shaders, which default VBSP does not support.
* Mapbase uses parallax corrected cubemaps, which require special changes to support.

Mapbase makes its own changes to the map compilers for this, although other community tools have also been created that can be used with Mapbase as well. This includes VBSP++ and Slammin' Source's VBSP, although the latter does not support parallax corrected cubemaps.
